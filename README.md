 Intelligent Data Analyzer — Upload any CSV and get instant AI-powered insights, anomaly detection, AutoML pipeline, and auto-generated reports. Built with Python, Scikit-learn, XGBoost, LightGBM, and Streamlit.


#  Intelligent Data Analyzer

> Upload any CSV and get instant AI-powered insights!


##  Features
- ✅ Automated Data Cleaning & Profiling
- ✅ Interactive Visualizations (Plotly)
- ✅ Anomaly Detection (IsolationForest + LOF)
- ✅ AI Feature Importance Suggestions
- ✅ AutoML Pipeline (XGBoost, LightGBM, Random Forest)
- ✅ Auto-generated HTML + PDF Reports

##  Tech Stack
- **Data:** Pandas, NumPy
- **Visualization:** Matplotlib, Seaborn, Plotly
- **ML:** Scikit-learn, XGBoost, LightGBM
- **Stats:** Scipy, Statsmodels
- **App:** Streamlit

##  How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
```
